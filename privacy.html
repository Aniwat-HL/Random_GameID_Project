นโยบายความเป็นส่วนตัว

วันที่มีผลบังคับใช้: [ระบุวันที่]

นโยบายความเป็นส่วนตัวนี้อธิบายถึงวิธีการที่เรารวบรวม ใช้ และปกป้องข้อมูลของคุณเมื่อคุณใช้แพลตฟอร์มเกมของเรา โดยเฉพาะการใช้ Google Email สำหรับการเข้าสู่ระบบเพื่อรับหมายเลข ID สำหรับเล่นเกมโดยที่เราไม่เก็บหรือใช้ข้อมูลส่วนตัวของคุณในทางอื่น

1. ข้อมูลที่เรารวบรวม

  • Google Email: เราจะเก็บอีเมลที่ใช้สำหรับการเข้าสู่ระบบผ่าน Google เพื่อให้คุณสามารถลงทะเบียนและรับหมายเลข ID เกมที่ไม่ซ้ำกัน

2. วิธีการที่เราใช้ข้อมูลของคุณ

  • การเข้าสู่ระบบและการจัดการบัญชีผู้ใช้: ใช้ Google Email เพื่อเข้าสู่ระบบและให้คุณได้รับหมายเลข ID สำหรับเล่นเกม
  • การจัดการ ID เกม: เพื่อให้คุณได้รับหมายเลข ID ที่ไม่ซ้ำกันสำหรับการเล่นเกมของคุณ

3. ข้อมูลที่เราไม่ใช้

เราไม่ใช้ข้อมูลส่วนตัวของคุณ นอกจากที่กล่าวถึงในส่วนของการเข้าสู่ระบบและการจัดการ ID เกม

4. ความปลอดภัยของข้อมูล

เราให้ความสำคัญกับความปลอดภัยของข้อมูลของคุณ โดยใช้มาตรการที่เหมาะสมในการปกป้องข้อมูลที่เก็บรักษา:

  • ข้อมูลอีเมลของคุณจะได้รับการเก็บรักษาอย่างปลอดภัยและใช้เฉพาะในการเข้าสู่ระบบ
  • เราไม่เปิดเผยข้อมูลส่วนตัวของคุณให้กับบุคคลภายนอก
